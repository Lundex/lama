Tables

PlayerState
Singleton that contains enum-styled values for Player.state.

Fields:

  • NEW: Player has just been connected.
  • DISCONNECTING: Player is in the process of disconnecting.
  • NAME: Retreiving a name to be used by the player.
  • NEW_CHAR_NAME_CONFIRM: Confirming they wish to use this name to create a new character.
  • NEW_CHAR_PASSWORD: Asking for a password for their new character.
  • NEW_CHAR_PASSWORD_CONFIRM: Asking to confirm their password.
  • LOAD_CHAR_PASSWORD: Asking for the password for an existent character.
  • MOTD: Getting the MOTD.
  • PLAYING: Playing the game.
PlayerState.names
Contains textual representations of PlayerState enums.

Fields:

  • PlayerState.NEW: "new"
  • PlayerState.DISCONNECTING: "disconnecting"
  • PlayerState.NAME: "name"

  • PlayerState.NEW_CHAR_PASSWORD: "(new char) password"
  • PlayerState.NEW_CHAR_PASSWORD_CONFIRM: "(new char) password confirm"
  • PlayerState.LOAD_CHAR_PASSWORD: "(load char) password"
  • PlayerState.MOTD: "MOTD"
  • PlayerState.PLAYING: "PLAYING"

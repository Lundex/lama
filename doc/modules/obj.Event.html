Event:initialize (destination, fun, shouldRepeat, repeatMax, repeatInterval)
This is just a shortcut for initializing a new event. This is to make it as easy as possible to create a new event.

Parameters:

  • destination: Timestamp indicating when the event should fire.
  • fun: The function to be fired by the event.
  • shouldRepeat: If true, this event should repeat after the first firing.
  • repeatMax: How many cycles before we stop.
  • repeatInterval: How long between each repeat? (relative amount based on clock)

Event:willRepeat ()
Will this event repeat (anymore)? Takes into consideration whether or not we have reached our repeat maximum, so it will return false if we have reached it.

Return value:

    true if it will repeat.
    false otherwise.

Tables

Event
Event meant to be used with a Scheduler.

Fields:

  • destination: Timestamp indicating when the event should fire.
  • didRun: If true, this event has had its initial firing.
  • shouldRepeat: If true, this event should repeat after the first firing.
  • currentRepeat: Which cycle we're on.
  • repeatMax: How many cycles before we stop.
  • repeatInterval: Added to destination for each repeat.

CharacterManager.characterNameTaken (name)
Check if a character name is in use.

Parameters:

  • name: Name to check.

Return value:

    true if name is taken.
    false otherwise.
CharacterManager.generateCharacterData (mob)
Get the XML format of a character.

Parameters:

  • mob: Mob of the character to generate data of.

Return value:

    The XML format of the character.
CharacterManager.getCharacterFileFromName (name)
Get the character filename for a given name.

Parameters:

  • name: Name of the character to generate a filename for.

Return value:

    Properly formatted filename for character based on its name.
CharacterManager.legalizeName (name)
Get the legalized name form of the given string.

Parameters:

  • name: Name to be legalized.

Return value:

    The legalized name, or nil followed by an error message.
CharacterManager.loadCharacter (name, mob)
Load a character into a mob.

Parameters:

  • name: Name of the character to be loaded.
  • mob: The mob for character data to be read into.
    if nil, a new mob will be generated and returned.

Return value:

    The mob that data was read into.
CharacterManager.readCharacterData (xml, mob)
Read XML data into a mob's attributes.
I am not liking the Expat library. I might switch to LuaXML.

Parameters:

  • xml: The XML to be read.
  • mob: Optional mob to read data into.
    If not specified, will return a new mob created with the XML data.

Return value:

    The mob that the data was applied to.

Save a mob as a character.

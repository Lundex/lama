Tables

MapObject
Cloneable that is used to represent objects on a Map.

Fields:

  • name: Name of the MapObject.
  • description: Complete description of the MapObject.
  • map: Map that the MapObject is a part of.
  • loc: MapObject that the MapObject inhabits.
MapObject.contents
Contains all of the MapObjects inhabiting this MapObject.

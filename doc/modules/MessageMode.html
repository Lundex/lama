Tables

MessageMode
Singleton that contains enum-styled values for Player.messageMode.

Fields:

  • GENERAL: General messages.
  • CHAT: Character communication.
  • ANNOUNCEMENT: Critical game updates.
  • INFO: Informational updates that don't fit in other categories. e.g. spell fizzling.
  • QUESTION: Mode for questions. Questions generally don't end with a linebreak, so this might be useful in the future.
  • MOVEMENT: Movement updates.
  • COMBAT: Combat updates.
  • FAILURE: Error messages.

Server:initializeClientSocket (socket)
Initialize the socket's settings for an incoming Client. By default, the Client socket is made to act asynchroniously, with a 1/1000th second timeout for I/O operations.

Parameters:

  • socket: The socket to initialize.
Server:initializeServerSocket (socket)
Initialize the socket's settings for this Server. By default, the Server socket is made to act asynchroniously, with a 1/1000th second timeout for I/O operations.

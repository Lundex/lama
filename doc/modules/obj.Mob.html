Tables

Mob
Cloneable:MapObject that holds data for mobile creatures.

Fields:

  • name: Name of the creature.
  • description: A complete description of the creature.
  • level: Experience level of the creature.
  • experience: Experience accumulated this level.
  • health: Current health.
  • mana: Current mana.
  • moves: Current moves.
  • player: The Player we're associated with.

Tables

Cloneable Table that provides psuedo-inheritance-type features for tables.

Inheritance is achieved by using the __index metatable method, which just transfers attempts to index nil fields to the parent.
  • Cloneable.clone(parent) provides an interface for making a pure clone that is set to refer to the given parent.
  • Cloneable.new(parent, ...) provides an interface for making an instance-style clone, which has the properties of a pure clone, but is initialized.
In effect, pure clones are meant to act as a class of a sort, and instance-style clones are meant to act as instances of a class.

Cloneable.isCloneOf (clone, ancestor)
Traverses a clone's parentage via their metatable's __index value. Determines whether or not this clone is a clone of another Cloneable.

Parameters:

  • clone: The Cloneable to check.
  • ancestor: The Cloneable we might be a clone of.

Return value:

    true if given clone is a clone of given ancestor.
    false otherwise.
